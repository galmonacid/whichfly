# Roadmap – whichFly

## MVP
- Core “Right now” recommendation flow
- GPS-based river suggestion
- UK rivers – trout
- Full UK river dataset with reach-level points for context (GB + NI)
- Flutter client for iOS + web (shared UI codebase)

---

## Post-MVP
- Planning mode improvements
- Water level automation (selected rivers)
- Feedback loop (“Did it work?”)
- Seasonal passes / monetisation
- Finer reach/beat selection (if needed)

---

## Future ideas
- Club and fishery integrations
- Region-specific tuning
- Offline-first support
- Android target from the same Flutter codebase

---

## Explicitly discarded ideas
- Social feeds
- Ads
- Sponsored fly placement
